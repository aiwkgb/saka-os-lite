# SAKA OS Lite

Open-source operating template for solo operators, freelancers, and Indonesian UMKM operators who want an AI-assisted daily command center.

## Live demo

After GitHub Pages is enabled:

https://aiwkgb.github.io/saka-os-lite/

## What it includes

- Project Radar
- Daily BRIEF and LOG templates
- Content Pipeline
- Ads/metrics watchlist
- Opportunity and bounty tracker
- Static dashboard with demo data

## Quick start

Run from repo root:

    python3 scripts/validate.py
    python3 scripts/generate_brief.py
    python3 scripts/generate_log_template.py
    python3 scripts/sync_dashboard_data.py

Then open dashboard/index.html.

## Privacy first

This public template contains only demo data. Do not commit real client names, secrets, ad account IDs, phone numbers, or private leads.
