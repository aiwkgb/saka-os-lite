# Contributing

Thanks for your interest.

## Good first contributions

- Improve documentation clarity.
- Add safe demo examples.
- Fix typos.
- Improve accessibility and mobile layout.
- Add deployment notes.

## Privacy rules

Do not include private client data, phone numbers, emails, API keys, ad account IDs, or screenshots from real accounts unless you have explicit permission.

## Development

Run the validation commands in the README before opening a pull request.
