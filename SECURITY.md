# Security Policy

SAKA OS Lite is a public-safe template. It is not designed to store secrets or private operational records.

## Supported versions

The current `main` branch is supported for documentation and demo-template fixes.

## Please do not commit

- API keys, tokens, passwords, private keys
- real client/customer data
- phone numbers, emails, invoices, legal documents
- ad account IDs or screenshots from real business accounts
- personal/family/finance data

## Reporting a security issue

Open a GitHub issue with a minimal description if the issue does not expose sensitive data.

If a report contains sensitive information, do not post it publicly. Contact the maintainer privately via the GitHub profile contact path.

## Design safety

This repo intentionally uses:

- local JSON files,
- static dashboard files,
- no backend,
- no default external API calls,
- no automatic public posting,
- no ads/account integrations.
