# SAKA OS Lite Roadmap

This roadmap keeps the public template small, useful, and safe.

## v0.1 — Public foundation

Status: done.

- Demo JSON data
- Daily BRIEF generator
- LOG template generator
- Dashboard sync script
- Static dashboard
- Sanitization checklist
- MIT license

## v0.2 — Portfolio polish

Status: current.

- Better README positioning
- Beginner quickstart
- Why/roadmap docs
- Screenshot guidance
- Expanded privacy notes
- Release notes
- Public proof readiness

## v0.3 — Better demo experience

Planned:

- Improve dashboard layout and mobile readability
- Add sample screenshots or GIF using demo data only
- Add guided tutorial: "Run your first operating loop"
- Add more demo opportunities/content examples
- Add basic accessibility checks

## v0.4 — Template packs

Planned:

- Solo freelancer template
- UMKM operator template
- Content creator template
- AI builder template
- Optional CSV import/export examples

## v0.5 — Agent-ready workflow examples

Planned, not automatic:

- Example prompts for generating a Daily BRIEF from data
- Example prompt for turning LOG into next actions
- Example prompt for reviewing Project Radar
- Clear warning that users should not send sensitive data to AI tools without understanding privacy risks

## Non-goals for now

- No login system
- No hosted backend
- No paid API dependency
- No real customer data
- No automatic public posting
- No ads account integration
- No finance/legal/private vault

## Guiding principle

Keep SAKA OS Lite useful for low-resource builders:

> Simple enough to copy. Structured enough to become useful. Safe enough to publish.
